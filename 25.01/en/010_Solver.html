Solver (advanced settings for detailed adjustments)

For users who want to further fine-tune the analyzed animation output results. You can freely customize various settings as needed.

More advanced topics including benchmark results are available here。

Solver

▼Global: allows you to change the settings for analysis.

  • ☑ Fisheye: check this box to optimize for analyzing videos shot with a fisheye lens.

Note

The checkbox is already marked in a default setting. Please unmark this checkbox when analyzing videos shot with a wide-angle lens.

  • ▼Smoothing: allows you to change the settings for applying smoothing processing during animation output. Each region can be changed individually.

    • Smoothing: whether smoothing processing is applied or not (Default to True).

    • Weight: strength of the smoothing process per output. The higher the value, the stronger the smoothing process (smoother animation curves).

    • Reps: number of times the smoothing process is applied. The higher the number, the smoother the animation curve.

    • Save current smoothing profile：スムージング設定の保存 保存したスムージング設定はボタンの下に表示されます(クリックで反映、右クリックでメニュー表示)

▼Post processing: allows you to change settings such as clamp processing during animation output.

  • Prioritize profile：アニメーション出力範囲内にProfileを作成したフレームが含まれる場合、チェックを入れると解析による予測値よりもリターゲット値を優先する

  • Clamp mode: toggles clamp processing settings.

    • No Clamp: no clamp processing

    • Hard Clamp: clamps the animation curve to the maximum and minimum values set in the Controller window (no smoothing of the clamped area).

    • Soft Clamp: clamps the animation curve at the maximum and minimum values set in the Controller window (with smoothing of the clamped range).

  • Train：ユーザーが変更したprofile設定を参照し、Solverを再構築します。
    Solverは動画出力やアニメーション出力などを行う度に自動で再構築されるため、基本的には使用する必要はありません。

  • Delete cache：ユーザーが変更する前のprofile設定で構築した解析キャッシュを削除します。

  • Export：リアルタイムアニメーション用のソルバーを出力します。
