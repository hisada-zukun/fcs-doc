Prerequisite

FCS is compatible with computers with the following specifications.

Requirement

Recommendation

CPU

Intel Core i5-9400F

Intel Core i7-13700K

GPU

-

Nvidia RTX 3080

System Memory

DDR4 16GB+

DDR5 64GB+

Software

  • Required

    • Maya (Compatible with: 2018, 2020, 2022, 2023, 2024)

  • Recommendation

    • Nvidia CUDA/CuDNN

License Obtaining a license key is required to use FCS. We offer a one-month free trial license. If you’re interested, please contact us from the HELTEC website.
